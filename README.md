# Product Analytics & dbt Engineering

Analytics Engineering project using Snowflake, dbt, and GA4 ecommerce event data.

## Project Goals

- Build an end-to-end analytics engineering workflow
- Model ecommerce event data
- Create dimensional models
- Analyze user conversion funnels
- Measure customer retention
- Build analytics-ready marts

## Tools

- Snowflake
- dbt
- SQL
- Git
- GitHub
- BigQuery

Due to BigQuery Sandbox export limitations, a representative subset of event-level data was loaded into Snowflake for modeling. Product, user, and purchase source tables were fully loaded.
